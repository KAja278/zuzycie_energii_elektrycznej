# Energy Consumption Analysis

Projekt analizy zużycia energii elektrycznej w gospodarstwie domowym oraz budowy modeli predykcyjnych na podstawie danych czasowych.

## Technologie

R, ggplot2, dplyr, modele regresyjne

## Przebieg projektu

przygotowanie i czyszczenie danych
przeprowadzenie eksploracyjnej analizy danych (EDA)
budowa modeli predykcyjnych (regresja wieloraka, drzewo regresyjne)
wizualizacja danych i interpretacja wyników

## Funkcjonalności
analiza zużycia energii w czasie
identyfikacja zależności między zmiennymi
budowa i porównanie modeli predykcyjnych
wizualizacja wyników

## Uruchomienie
install.packages(c("ggplot2", "dplyr"))

source("CS.R")

## Opis działania

Dane dotyczą zużycia energii elektrycznej w czasie (time series). W projekcie przeprowadzono analizę zależności między zmiennymi oraz zbudowano modele regresyjne umożliwiające przewidywanie poziomu zużycia energii.

## Dane

Zbiór danych zawiera informacje o zużyciu energii, napięciu oraz natężeniu prądu w różnych przedziałach czasowych.
